10.  Why do we use media queries?

Media queries are used in stylesheets and they allow browsers to run tests that determine the size of a window in order to display media selectively.
For example, a media query might be used to ensure that an image is remaining fit to the screen, even when the screen itself is getting smaller or larger.


11. What is the main difference between a serif and a sans-serif font?

The main difference between serif and sans-serif font is that the edges of serif font include decoration, while the edges of sans-serif font have no decoration.
